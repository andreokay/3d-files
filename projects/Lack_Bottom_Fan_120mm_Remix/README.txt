# Summary

Just a quick remix on Xander23669's Ikea Lack Fan to make it big enough for 120mm fans (their original was an 80).  Did a simple 60mm expansion in Tinkercad.

I've got a Maker Select Plus and although so far no problems with PLA and 1 exhaust fan I wanted to put an intake directly below the power supply and Meltzi board to keep them cool.  Side effect is cooling the enclosure when I need it if I need to print PLA instead of PETG or ABS.
